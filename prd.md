# WTC Membership App PRD

## 1. 프로젝트 개요

WTC Membership App은 WTC SEOUL 홈페이지의 기존 입주사 전용 기능과 관리자페이지 기능을 모바일 앱으로 확장하고, 입주사 전용 혜택, 코엑스 전시 일정, 공지사항, 쿠폰, 커뮤니티 기능을 통합 제공하는 입주사 생활 플랫폼이다.

본 앱은 단순한 건물관리 앱이 아니라, 입주사가 업무 신청을 처리하고, WTC 멤버십 혜택을 확인하며, 코엑스 전시 일정과 건물 공지를 받아보고, 입주사 간 정보 교류까지 할 수 있는 앱을 목표로 한다.

## 2. 핵심 컨셉

> WTC에서 일하는 입주사를 위한 업무·혜택·정보·커뮤니티 통합 멤버십 앱

앱의 주요 방향은 다음과 같다.

1. 기존 WTC SEOUL 홈페이지 기능을 앱으로 확장
2. 입주사 전용 메뉴 및 마이페이지 연동
3. WTC 멤버십 혜택 제공
4. 코엑스 전시 일정 API 연동
5. 건물 공지 및 GITC 공사 안내 제공
6. 고객센터 번호 연결을 통한 간편 민원 문의
7. 입주사 간 제휴/홍보/정보 공유 커뮤니티 제공

## 3. 제외 기능

아래 기능은 현재 범위에서 제외한다.

- 관리비 조회
- 임대료 조회
- 출입/보안 시스템 연동
- 라운지 예약
- 공용공간 예약
- 멤버십 등급제
- 입주사 기업 소개관 형태의 별도 소개 페이지
- 민원/VOC 처리 시스템
- 전자결재 연동

민원은 별도의 접수/처리 시스템이 아니라, 고객센터 번호 연결 또는 기존 고객의 소리 메뉴로 연결한다.

# 6.1 홈

홈 화면은 입주사가 앱을 열었을 때 오늘 필요한 정보를 한눈에 볼 수 있는 대시보드 역할을 한다.

## 주요 구성

- 오늘의 공지
- GITC 공사 안내
- 오늘의 혜택
- 코엑스 전시 일정
- 주차 만차 여부
- 빠른 신청
- 고객센터 바로 연결
- 주요 알림

## 홈 화면 예시

- 오늘의 WTC
- 공지 3건
- GITC 공사 안내 2건
- 오늘의 혜택 5건
- 진행 중인 코엑스 전시 4건
- 주차 상태: 여유 / 혼잡 / 만차

## 홈 카드 예시

### 오늘의 공지
- [공사] GITC 공사 관련 이동 동선 안내
- [운영] 주차장 이용 안내
- [점검] 냉난방 설비 점검 안내

### 오늘의 혜택
- 코엑스몰 F&B 10% 할인
- 호텔 레스토랑 입주사 우대
- 쇼핑 브랜드 입주사 할인

### 코엑스 전시 일정
- 현재 진행 중인 전시
- 이번 주 전시
- 오늘 시작하는 전시

# 6.2 신청

기존 WTC SEOUL 홈페이지의 입주사 전용 메뉴와 마이페이지 기능을 앱으로 가져온다.

기존 홈페이지에 이미 있는 신청 기능은 앱에서 새로 만들기보다, 기존 홈페이지 DB/API와 연동하는 방식을 우선 검토한다.

## 주요 메뉴

- 방문자 사전신청
- 임시정차 신청
- 추가 냉난방 신청
- 화물엘리베이터 전용신청
- 회의실 대관신청
- 천장텍스 신청
- 고객의 소리
- 공사 및 작업 신청방법
- 입주사 공지게시판
- 입주사 마이페이지

## 신청 기능 공통 플로우

1. 입주사 로그인
2. 신청 메뉴 선택
3. 신청서 작성
4. 첨부파일 또는 비고 입력
5. 신청 완료
6. 마이페이지에서 신청 내역 확인
7. 관리자페이지에서 접수 확인

## 기존 홈페이지 연동 대상

WTC SEOUL 홈페이지의 입주사 전용 메뉴 및 마이페이지를 앱에서 접근 가능하도록 한다.

연동 대상 예시:

- 입주사 공지게시판
- 공사 및 작업 신청방법
- 화물E/V 전용신청
- 추가 냉난방 신청
- 임시정차 신청
- 방문자 사전신청
- 회의실 대관신청
- 천장텍스 신청
- 고객의 소리
- 개인정보 수정
- 입주사 회원관리
- 신청 내역 조회

## 구현 방식

우선순위는 다음과 같다.

1. 기존 홈페이지 API 연동
2. API가 없다면 신규 API 개발 요청
3. API 개발이 어려울 경우 WebView 임시 적용
4. 장기적으로 앱 전용 화면 및 통합 관리자페이지 구축

# 6.3 멤버십

멤버십 메뉴는 WTC 입주사 전용 혜택을 제공하는 핵심 메뉴다.

앱의 차별화 포인트이며, 입주사 직원들이 앱을 자주 열어보게 만드는 역할을 한다.

## 메뉴명

- WTC 멤버십

## 주요 카테고리

1. 오늘의 혜택
2. F&B 할인
3. 쇼핑 할인
4. 호텔 할인
5. 코엑스 전시/문화 혜택
6. 쿠폰함
7. 인기 혜택
8. 신규 혜택

## 혜택 상세 정보

각 혜택 상세 페이지에는 아래 정보를 표시한다.

- 혜택명
- 브랜드명
- 카테고리
- 할인율 또는 제공 혜택
- 사용 기간
- 사용 가능 지점
- 사용 방법
- 유의사항
- 입주사 인증 필요 여부
- 쿠폰 다운로드 버튼
- 지도 또는 위치 안내
- 문의 연락처

## 혜택 카테고리 예시

### F&B

- 코엑스몰 식당 할인
- 카페 할인
- 베이커리 할인
- 점심 메뉴 할인
- 호텔 레스토랑 할인

### 쇼핑

- 코엑스몰 매장 할인
- 라이프스타일 브랜드 할인
- 뷰티/패션 브랜드 할인
- 서점/문구 할인

### 호텔

- 호텔 객실 우대가
- 호텔 뷔페 할인
- 호텔 레스토랑 할인
- 라운지 할인
- 베이커리 할인

### 전시/문화

- 코엑스 전시 할인
- 전시 초대권
- 문화행사 할인
- 공연 할인

## 쿠폰 상태

- 사용 가능
- 사용 완료
- 기간 만료

## 쿠폰 기능

- 쿠폰 받기
- 쿠폰 저장
- 쿠폰 사용하기
- QR 또는 바코드 표시

- 사용 조건 표시

# 6.4 코엑스 전시 일정

코엑스 전시 일정은 코엑스 공식 사이트에서 API를 요청하여 연동한다.

연동 대상 URL:

https://www.coex.co.kr/event/full-schedules/#;

## 주요 기능

- 오늘 진행 중인 전시
- 이번 주 전시
- 이번 달 전시
- 전시 상세 정보
- 전시 장소
- 전시 기간
- 공식 페이지 링크
- 관심 전시 저장
- 전시 시작 알림

## 전시 정보 필드

- 전시명
- 시작일
- 종료일
- 장소
- 주최/주관
- 전시 카테고리
- 관람 시간
- 입장료
- 상세 설명
- 대표 이미지
- 공식 URL
- 입주사 할인 여부

## 구현 방식

1. 코엑스 측에 전시 일정 API 제공 가능 여부 확인
2. API 제공 가능 시 앱 서버에서 일정 데이터 수집
3. 앱에서 날짜별, 카테고리별로 노출
4. API 제공이 어렵다면 크롤링이 아닌 공식 데이터 제공 방식 협의 필요
5. 단기적으로는 관리자페이지에서 전시 일정을 수동 등록할 수 있도록 구성

## 코엑스 API 요청 시 확인할 사항

- 전시 일정 API 제공 여부
- API 제공 방식
- 인증키 필요 여부
- 호출 제한
- 이미지 사용 가능 여부
- 상세 페이지 링크 사용 가능 여부
- 상업적/서비스 이용 가능 여부
- 데이터 갱신 주기
- 전시 취소/변경 정보 반영 여부

# 6.5 공지사항

공지사항은 너무 세분화하지 않고 3개의 큰 카테고리로 운영한다.

## 공지 카테고리

### 1. 운영공지

건물 운영과 관련된 일반 공지.

포함 내용:

- 일반 공지
- 주차 안내
- 이용 안내
- 입주사 안내
- 행사 안내
- 서비스 운영 안내

### 2. 공사·점검공지

공사, 점검, 소음, 동선 변경 등과 관련된 공지.

포함 내용:

- GITC 공사 안내
- 공사 일정
- 작업 안내
- 정기점검
- 설비 점검
- 엘리베이터 점검
- 전기/소방/냉난방 점검
- 통행 제한 안내

### 3. 긴급공지

즉시 확인이 필요한 공지.

포함 내용:

- 긴급 안전 안내
- 사고 안내
- 시설 장애
- 정전/단수
- 주차장 만차
- 긴급 통행 제한
- 긴급 푸시 알림

## 공지 상세 정보

- 제목
- 카테고리
- 작성일
- 적용 기간
- 대상 건물
- 내용
- 첨부파일
- 이미지
- 문의 연락처
- 푸시 발송 여부

## GITC 공사 공지 필드

GITC 공사 관련 공지는 자주 발생할 수 있으므로 아래 필드를 권장한다.

- 공사명
- 공사 기간
- 작업 시간
- 작업 위치
- 소음 발생 여부
- 통행 제한 여부
- 주차 영향 여부
- 이용자 유의사항
- 문의 연락처
- 첨부 도면 또는 이미지

# 6.6 주차

주차 기능은 상세한 주차 관리 기능이 아니라, 현재 주차장 만차 여부를 확인하는 수준으로 구현한다.

## 주요 기능

- 주차 상태 확인
- 만차 여부 표시
- 주차 관련 공지 확인
- 주차장 안내 바로가기

## 주차 상태 값

- 여유
- 혼잡
- 만차
- 정보 없음

## 표시 예시

- 현재 주차 상태: 여유
- 현재 주차 상태: 혼잡
- 현재 주차 상태: 만차

## 구현 방식

1. 주차 시스템 API를 통해 만차 여부 수신
2. API가 없을 경우 관리자 수동 입력 방식 제공
3. 홈 화면에 간단히 표시
4. 주차 메뉴에서는 주차 안내와 관련 공지를 함께 제공

# 6.7 민원 / 고객센터

별도의 민원 처리 시스템은 구축하지 않는다.

민원 기능은 고객센터 번호 연결 방식으로 간단하게 구현한다.

## 주요 기능

- 고객센터 전화 연결
- 담당 부서별 연락처 안내
- 고객의 소리 바로가기
- 분실물 문의 연결
- 주차 문의 연결
- 시설 문의 연결

## 메뉴명 후보

- 고객센터
- 문의하기
- 민원 연락처
- 담당자 연락처

## 고객센터 화면 구성

- 대표 고객센터 번호
- 시설 문의
- 주차 문의
- 냉난방 문의
- 방문자 신청 문의
- 공사/작업 문의
- 분실물 문의
- 고객의 소리 바로가기

## 동작 방식

사용자가 전화번호를 누르면 즉시 전화 앱으로 연결한다.

예시:

- tel:02-0000-0000

# 6.8 커뮤니티

커뮤니티는 입주사 간 자유로운 정보 공유와 제휴 제안을 위한 메뉴다.

입주사가 직접 게시글을 작성할 수 있게 한다.

## 커뮤니티 메뉴

1. 채용/홍보
2. 중고물품 나눔
3. 사내행사 홍보
4. 입주사 간 제휴 제안

예시:

- 입주사 전용 혜택 안내
- 신제품 소개
- 브랜드 캠페인 소개
- B2B 서비스 제안
- 임직원 대상 할인 안내
